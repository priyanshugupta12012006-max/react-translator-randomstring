# React Translator & Random String Generator

A simple React project that translates text using LibreTranslate API and generates random strings.

## Features
- Translate text between English, Hindi, and other languages
- Random string generator
- Clean UI with Tailwind CSS
- Easy to use

## Tech Stack
- React
- Tailwind CSS
- LibreTranslate API
- Vite
